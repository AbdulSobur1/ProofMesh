# 🧠 ProofMesh

**ProofMesh is a decentralized proof-of-skill platform where users submit real work, get AI-verified evaluations, and build a verifiable reputation layer for the internet.**

---

## 🚀 The Problem

In today’s digital world, **skills are easy to claim but hard to verify**.

- Resumes are inflated  
- Portfolios can be misleading  
- Recruiters rely on trust instead of proof  

This creates a system where:
> ❌ Claims > Actual ability  

---

## 💡 The Solution

**ProofMesh flips the model: Proof > Claims**

Users submit real work — projects, code, or explanations — and:

1. 🤖 **AI evaluates the submission**
2. 📊 Generates a **skill score + feedback**
3. 🔗 Stores it as a **tamper-proof record (simulated on-chain)**
4. 🧠 Builds a **dynamic reputation profile**

---

## ⚙️ How It Works

### 1. Submit Proof
Users provide:
- Project title
- Description
- Link or explanation

---

### 2. AI Verification
The system analyzes:
- Technical quality  
- Clarity of explanation  
- Relevance of skills  

Returns:
- **Score (0–10)**
- **Feedback**
- **Skill tags**

---

### 3. Proof Record
Each submission gets:
- Unique ID  
- Timestamp  
- Simulated transaction hash  

> Designed for future on-chain verification

---

### 4. Reputation System
Users build:
- Aggregate skill score  
- Verified proof history  
- Tag-based skill profile  

---

### 5. Public Profile
A shareable profile where:
- Skills are **verified, not self-declared**
- Recruiters can instantly assess ability
